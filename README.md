# Label-Encoder-ML

## 📌 Description

A simple machine learning project demonstrating Label Encoding on categorical data using scikit-learn.

---

## 📌 What is Label Encoding?

Label Encoding converts categorical values into numeric labels by assigning a unique number to each category.

---

## ⚙️ Workflow

* Import required libraries (pandas, scikit-learn)
* Create dataset using Python dictionary
* Convert dictionary into DataFrame
* Apply LabelEncoder from Scikit-learn
* Convert categorical values into numerical labels


---

## 🚀 Features

* Converts categorical data into numerical labels using LabelEncoder
* Creates dataset using Python dictionary
* Uses Pandas for data handling
* Simple and beginner-friendly implementation
* Clean and easy-to-understand workflow

---

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn (`LabelEncoder`)
* Jupyter Notebook
